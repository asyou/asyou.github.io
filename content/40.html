Python将latex转换成Mathml

2022-08-19

在PHP中要将latex数学公式转换成Word文档所需的Mathml非常困难，几乎没有找到相应的插件或代码，但在Python中是非常简单的事情。

使用Python转换latex为Mathml需要用到一个插件：latex2mathml，在Github或Gitee上都能找到。

一、pip安装latex2mathml

在命令行中输入以下命令进行安装

pip3 install latex2mathml

需要注意，如果本地的是pip的话要把pip3改成pip。

二、运行并测试

在终端中进入Python环境进行测试

wangwei@Mac ~ % python3
Python 3.9.6 (default, Jun 29 2021, 06:20:32) 
[Clang 12.0.0 (clang-1200.0.32.29)] on darwin
Type "help", "copyright", "credits" or "license" for more information.
>>> import latex2mathml.converter
>>> latex = '$y=-3(x+2)^{2}-5$'
>>> mathml = latex2mathml.converter.convert(latex)
>>> print(mathml)
<math xmlns="http://www.w3.org/1998/Math/MathML" display="inline">
  <mrow>
    <mi>$</mi>
    <mi>y</mi>
    <mo>&#x0003D;</mo>
    <mo>&#x02212;</mo>
    <mn>3</mn>
    <mo stretchy="false">&#x00028;</mo>
    <mi>x</mi>
    <mo>&#x0002B;</mo>
    <mn>2</mn>
    <msup>
      <mo stretchy="false">&#x00029;</mo>
      <mrow>
        <mn>2</mn>
      </mrow>
    </msup>
    <mo>&#x02212;</mo>
    <mn>5</mn>
    <mi>$</mi>
  </mrow>
</math>
>>>

可见，在Python中转换latex为Mathml是非常方便的（终端中输出的mathml是单行显示，为方便阅读我整理了一下格式）！复制Mathml到Word文档中粘贴即自动显示为公式了！
