uniapp上传并识别二维内容

2023-09-08

最近公司项目有这么一个需求，在移动端H5个人推荐信息中添加个人微信二维码，用于在分享的内容中，他人可以快速地识别二维码添加好友。

我们的想法是由用户自行上传个人微信中的二维码截图，但并不是直接把上传的带有二维码的图片展示出来，因为那样的话，不好看！

如果能在上传之后自动读取二维内容，并在移动端中使用类似Qrcode的二维码组件重新生成个人微信二维码，则能实现风格简洁统一的分享内容。

想到就干，首先想到的是用户上传二维后把图片传回后台，由后台进行二维码内容的识别，还真有方法，找到了一个qr-reader插件可以用。

一、安装插件

composer require zxing/qr-reader

二、调用插件并识别二维内容

// test.php
$qrcode = new \Zxing\QrReader('个人微信二维码图片');
$text = $qrcode->text();

这样即可正确识别二维码中的内容，先保存到用户表中，用于在移动端中生成对应的二维码。

但本着不死不休的强迫症习惯，我觉得由用户上传图片到服务器的方式并不好，会因此造成大量的垃圾图片，况且上传图片也会占用大量的服务器带宽，就算是识别之后立即删除图片也觉得难受。

就琢磨着有没有直接在移动端就完成二维码图片内容的识别方法，这一找还真有，不过大多都是什么调用uniapp的摄像头组件去扫描二维码，不符合要求。

我想要的是用户上传二维码图片后，直接通过某种方式识别出二维内容，而不需要调用摄像头（因为H5不支持uni.scanCode）。

结果还真找到一个组件，就是reqrcode.js，下载过来丢到项目中。需要注意的是如果是以这种下载的方式，下载完后需要修改一下源文件。

三、修改源文件

// reqrcode.js
module.exports = qrcode

在文件尾部添加一个暴露方法。

四、引用插件

// test.vue
import reqrcode from '@/utils/reqrcode.js'    // 二维码识别
import uQRCode from '@/uqrcode/uqrcode.js'    // 二维码生成

五、模板代码

// test.vue
<canvas id="qrcode" canvas-id="qrcode" class="canvas-hide" style="width:200px;height:200px;" />

六、JS代码

// test.vue
let _this = this
reqrcode.decode('上传图片后得到的临时文件路径')
reqrcode.callback = function(res) {
    if (res == 'error decoding QR Code') {
        _this.$api.msg('识别失败，请重试')
    } else {
        // 生成二维码
        var qr = new uQRCode()
        qr.data = res
        qr.size = 200
        qr.make()
        var canvasContext = uni.createCanvasContext('qrcode', _this)
        qr.canvasContext = canvasContext
        qr.drawCanvas()
        setTimeout(() => {
            uni.canvasToTempFilePath({
                canvasId: _this.canvasId,
                fileType: _this.fileType,
                width: _this.canvasWidth,
                height: _this.canvasHeight,
                success: (rep) => {
                    _this.fileList1 = [{
                        status: 'success',
                        message: '上传成功',
                        url: rep.tempFilePath
                    }]
                    uni.hideLoading()
                },
                fail: (err) => {
                    _this.$api.msg('识别失败')
                }
            })
        }, 300)
    }
}

七、CSS代码

// test.vue
.canvas-hide {
    position: fixed;
    right: 100vw;
    bottom: 100vh;
}

完成这些后，就实现了用户上传微信二维码图片，并自动识别出二维码文本内容，识别出来后通过接口返回到后端保存即可。
