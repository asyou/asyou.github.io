PHPQrcode生成二维码

2022-02-22

利用PHP生成二维码是很常用的功能，用于支付、分享海报等。

本文要实现的功能及流程

1、后台部署phpQrcode插件，实现API接口或任何需要的地方调用；

2、创建好的二维码可以直接输出图片，也可以转成base64格式，适应不同需要。


第一步，下载phpQrcode插件

插件地址：http://sourceforge.net/projects/phpqrcode/，点击下载

下载好后解压，放入vendor目录。

第二步，在全局的common.php中创建2个函数

// /application/common.php
/**
 * 生成二维码
 * @param string $content 二维码内容
 * @param string $path 保存路径
 * @return string $file 二维码文件路径
 */
function createQrcode($content,$path)
{
    vendor('phpqrcode.phpqrcode');
    $errorCorrectionLevel = 'L';  //容错级别
    $matrixPointSize = 5;      //生成图片大小
    //生成二维码图片
    $prefix = './uploads';
    if (!file_exists($prefix.$path)) {
        $rs = mkdir($prefix.$path,0777,true);
    }
    $filename = uniqid().time().'.png';
    $fullpath = $prefix.$path.'/'.$filename;
    \QRcode::png($content, $fullpath , $errorCorrectionLevel, $matrixPointSize, 2);
    $QR = $fullpath;        //已经生成的原始二维码图片文件
    $QR = imagecreatefromstring(file_get_contents($QR));
    //输出图片
    // imagepng($QR, 'qrcode.png');
    // imagedestroy($QR);
    return substr($fullpath, 1, strlen($fullpath)-1);
}

/**
* 生成二维码以base64输出,
* $content 二维码内容
*/
function createQrcode64($content, $level = 'L', $size = 5, $margin = 2)
{
    Vendor('phpqrcode.phpqrcode');
    $QRcode = new \QRcode();
    ob_start(); // 在服务器打开一个缓冲区来保存所有的输出
    $QRcode->png($content,false,$level,$size,$margin);
    $imageString = base64_encode(ob_get_contents());
    ob_end_clean(); //清除缓冲区的内容，并将缓冲区关闭，但不会输出内容
    return "data:image/jpg;base64,".$imageString;
}

在api接口中使用

// 该函数需要指定二维码图片存放目录，返回二维码图片存放路径
$data['qrcode'] = createQrcode('http://www.xxx.com/article/detail/id/1.html', '/qrcode');
// 该函数仅返回二维码的base64文件流，前端可直接用img渲染
$data['qrcode64'] = createQrcode('http://www.xxx.com/article/detail/id/1.html');
return $data;

说明：

两个函数的第一个参数都是二维码内容，可以是任意字符串内容；

两种方式生成二维码各有优劣，第一种适合二维码不变的情况，可以保存在服务器和数据库中长期重复使用，缺点是时间久了会生成大量二维码图片占用服务器硬盘空间；第二种直接输出文件流，适合不希望保存二维码的情况，也不占用硬盘空间，缺点是可能会轻微影响手机性能（如果性能太差的话）；

如果想在生成的二维码中嵌入logo，可以自行扩展。
