MAC安装Python3

2022-08-03

Mac自带了Python2.7版本，在终端中输入python后回车，输出类似下面的信息

Python 2.7.16 (default, Mar 25 2021, 03:11:28) 
[GCC 4.2.1 Compatible Apple LLVM 11.0.3 (clang-1103.0.29.20) (-macos10.15-objc- on darwin
Type "help", "copyright", "credits" or "license" for more information.
>>>

输入python命令默认进入python2的开发环境，如果同时安装了Python3.x版本，则可以使用python3命令进入python3的开发环境。

Python 3.9.6 (default, Jun 29 2021, 06:20:32) 
[Clang 12.0.0 (clang-1200.0.32.29)] on darwin
Type "help", "copyright", "credits" or "license" for more information.
>>>

如果没有安装Python3，则可以从其官网下载安装，Python官网：https://www.python.org

从官网找到macOS 64-bit installer版本下载，下载过来的文件是个.pkg包，执行并按引导安装即可。
